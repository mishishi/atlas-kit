# 百科图鉴信息图 · 主模板

> 用法：本主模板提供通用骨架与通用规则。使用时，根据主题类型，将对应的「分类模板」内容拼接在本模板之后一起使用（替换主视觉逻辑、8 个模块栏目、评分维度，并叠加该类专属防翻车规则）。最终整段发给图像模型。
> 主题词只需在下方「主题」处替换，具体内容由图像模型根据主题填充。

---

请生成一张高清竖版「百科图鉴信息图」。

Canvas / aspect ratio: 9:16.
The final image must use a strict 9:16 vertical aspect ratio.
Do not use 2:3, 3:4, 4:5, 1:2, 1:2.1, or any other vertical poster ratio.

目标视觉规格为 2160×3840 或更高，细节清晰，适合后续缩放与裁切。整体呈现为一本百科全书 / 博物图鉴中的一页知识档案。

## 画布比例 · 不可变更

生成高清竖版百科图鉴信息图。

画布比例必须严格为9:16。

禁止生成：
1. 正方形构图
2. 3:4比例
3. 4:5比例
4. 横版海报
5. 杂志跨栏版式

最终成品必须看起来像手机全屏浏览的 9:16 百科图鉴页。

主题：【填写主题】
类型：【城市 / 动物 / 宠物 / 植物 / 人物 / 节日 / 食物 / 历史事件 / 科技概念 / 器物 / 自然现象】

## 全局锚定 · 最高优先

用户第一眼必须认为：这是一本百科全书 / 博物图鉴中的一页知识档案。

而不是：海报、广告、宣传页、明信片、社交媒体配图。

整张图的一切——构图、用色、文案语气、信息密度——都要服务于「这是一页可收藏的知识档案」这个判断。任何让它看起来像营销物料的元素，都是错误。

## 整体视觉风格

设计语言：自然历史博物馆图鉴 + DK 百科全书 + 国家地理知识页。干净、可信、有收藏感。

- 背景：米白 / 暖奶油色纸张质感（接近 #F5F0E8），细微纸纹，不要纯白
- 排版：清晰的横向分区网格，留白适度但不空旷
- 字体：主标题与栏目名用衬线体；正文数据用无衬线体；数字与度量用等宽风格
- 强调色：由分类模板指定，全图统一使用同一色系，用于边框、图标、标签与高亮
- 边框：细分隔线（0.5–1pt）分隔各区；信息框为圆角
- 完整内页纸张外框：整张图必须有一个克制、清晰、低对比度的百科书页内页外框；外框位于 9:16 画布内部，只作为纸质档案页装帧，不得形成额外留白、拼接区或第二层页面
- 不要渐变，不要发光效果，不要重阴影
- 整体观感：博学、精确、可印刷、博物馆品质

## 页面结构 · 从上到下

### ① 标题区（顶部 10%）

- 大号衬线主标题：主题的中文名称（仅名称，不加「图鉴」「档案」等词）
- 较小斜体副标题：学名 / 外文名 / 别名
- 3–5 个关键词药丸标签，横向排列
- 一条细分隔线与下方主视觉分开
- 标题区可加入极淡的主题剪影或标本式小插图作为右上角背景装饰，透明度低、线稿感强，只增强百科书页氛围，不抢标题和标签

### ② 主视觉区（约 28–30%）

画面中上部放置精致、可信的主题主视觉。

**主视觉的具体形态与观察区逻辑由分类模板定义**——不同类型差异很大。本主模板只规定通用标准：精致、干净、结构清晰，博物图鉴插画质感，像知识档案中的核心样本；不过度夸张，不杂乱拼贴；不广告风，不 Q 版，不卡通海报风。

风格锚点（供图像模型参考）：自然历史博物馆标本绘图 / 博物学手绘 / 精细科普插画。冷静、克制、有观察感，而非戏剧化摄影大片。

观察框通用规则（具体观察什么由分类模板定义）：3–5 个观察框，有真实观察感、服务百科信息、不是游玩推荐标签；优先短标题，不用数字编号；不用跨区域标注线、不用箭头精确连接；靠空间位置暗示关联。

### ③ 信息模块区（中部 50%）

8 个信息模块，按 2 列 × 4 行网格排列（**8 个栏目名由分类模板定义**）。

每个模块包含：线性小图标 + 3–5 字中文栏目标题 + 2–4 条短信息（标签或「字段：数值」格式，不要整句长段落）。

所有模块视觉高度统一：相同框尺寸、相同内边距、相同字体层级，细边框，轻微色块背景。

空间不足时降级规则：先压缩每条信息字数 → 再合并相近模块（最低 6 个）→ 不得低于 6 个模块，不得缩小字号至难以阅读。

### ④ 可视化条带（约 7%）

一条横向条带，含 1–2 个紧凑可视化元素，优先选择非评分类图示（**具体选哪种由分类模板指定**）。若 8 个模块中已包含「快速评分卡」，本条带不要再次出现评分卡。

- 小型示意地图：地理分布或位置
- 简化时间轴：线上标 4–5 个关键节点圆点
- 结构图或剖面示意图
- 分布图、流程图、构成图或体型 / 规模对比图

### ⑤ 底部总结栏（底部 5%）

通栏横条，背景比正文区略深。含一句百科式总结（15–20 字，有判断、不空泛、非宣传口吻）+ 文字两侧 2–3 个相关线性小图标。

底部总结栏要像百科书页的精致收尾：可使用细分隔线、浅色底纹、左右小型剪影或印章式图标，形成收藏页的装帧感；装饰必须克制，不做成海报式口号区。

不出现「总结」「图鉴总结」「百科总结」等固定标题。

## 信息密度

信息丰富但不拥挤。不大面积留白，不密密麻麻小字。保持百科书页质感。

## 生成优先级

1. 主视觉准确清晰，标题明确，整体一眼像百科图鉴
2. 6–8 个完整信息模块同时存在，保持清楚的网格秩序
3. 栏目标题、关键词和短标签尽量清晰可读
4. 局部观察、地图、时间轴、结构图、流程图、构成图等可视化细节
5. 冷知识、小型补充模块和装饰性信息

空间不足时：先删第五优先级，再压缩第四，再减少正文条数。不取消完整模块，不靠缩小字号硬塞。

## 文字要求

中文简体。标题和栏目名必须清晰。正文以短句和标签为主。

所有中文字符须清晰、字形正确、完全可读。优先保证标题、栏目名、关键词标签可读。若空间不足，压缩句子，不缩小到不可读。

避免：长段落 · 极小字号 · 乱码 · 伪字 · 错别字 · 空泛宣传语 · 夸张广告口吻。

## 事实准确性

内容应基于常识性、稳定性较高的百科信息。

- 优先用约数、范围或定性描述（如「约九千余间」「体长 1 米左右」）
- 不生成看似精确但无法确认的数字（不编造精确到个位的统计值、具体到某年某日的争议性数据）
- 不确定的字段，用定性表达替代精确数值，宁可模糊勿造假
- 地图、评分、时间轴、风险等级均可为示意性表达

## 视觉规范

浅色背景，轻纸张质感。圆角信息框，细边框，轻阴影。统一线性图标风格，统一网格排版。高级知识图鉴风格。

## 全局防翻车规则（所有类型通用）

以下是所有主题共通的失败模式，无论哪一类都要避免（各分类模板还有自己特有的防翻车清单，叠加执行）：

- 不做成海报、广告、宣传拼贴、明信片、社交媒体配图
- 不用营销夸张词：「必游」「打卡」「网红」「爆款」「钜惠」「限时」「黑科技」「传奇」等
- 不要 logo、水印、二维码、品牌标识、扫码关注
- 不要戏剧化摄影大片光影（逆光、强氛围、电影感构图），主视觉要冷静克制
- 不要整段长正文，只用数据标签与短语
- 涉及人物 / 历史 / 争议主题，保持百科中立，不歌颂、不丑化、不煽动
- 不确定的数据一律用约数、范围或定性表达，宁可模糊勿造假

突出：知识整理 + 模块信息 + 图鉴式展示 + 收藏价值。
