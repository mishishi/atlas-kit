# 分类模板 · 植物

> 用法：接在主模板之后。用本模板的内容替换主模板中的主视觉逻辑、8 个模块栏目、评分维度，并叠加防翻车规则。

## 强调色

森林绿 `#4A8C5C`

## 固定 8 模块

1. 分类阶元
2. 形态描述
3. 生长周期
4. 分布产地
5. 生态需求
6. 用途价值
7. 毒性提示
8. 快速评分卡

## 评分维度

观赏价值 / 药用价值 / 食用价值 / 种植难度 / 生态重要性

## 观察区逻辑

植物用「器官分解观察」，这是植物图鉴的经典逻辑：叶（叶形叶脉）/ 花（花瓣雄蕊结构）/ 果实或种子（含横截面）/ 根系或茎干 / 表皮或刺毛细节。

用「叶」「花」「果实」「根」等短标题，不编号，不连线。横截面式剖面图很契合植物图鉴气质，可多用。

## 可视化条带

- 年度生长周期图（环形或线形），按月标出花期与果期
- 形态结构剖面：展示根、茎、叶、花、果等关键部位
- 地理分布小地图或生态关系示意图

## 一句话锚定

用户第一眼必须认为：这是一本植物志 / 本草图鉴里的物种页，像植物学家的腊叶标本与手绘解剖。

而不是：花店海报、园艺种草图、多肉摆拍。

## 防翻车规则

- 主视觉用博物手绘 / 腊叶标本风，精确呈现植株结构，不做唯美氛围摄影
- 不做成花店广告、园艺种草、绿植装饰美图
- 毒性提示是重要安全信息：有毒必须明确标注部位与等级（红色高亮），无毒注明「无毒」，不可省略不可含糊
- 用途不夸大药效，不写「包治百病」式伪医学宣称
- 花期果期用季节或月份范围，不编精确日期
- 分布只做大范围产地示意
