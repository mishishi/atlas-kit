# 分类模板 · 历史事件

> 用法：接在主模板之后。用本模板的内容替换主模板中的主视觉逻辑、8 个模块栏目、评分维度，并叠加防翻车规则。

## 强调色

古金 `#C08040`

## 固定 8 模块

1. 事件档案
2. 背景起因
3. 事件经过
4. 核心数据
5. 主要人物
6. 结果影响
7. 史学评价
8. 快速评分卡

## 评分维度

规模烈度 / 持续时长 / 波及范围 / 历史转折性 / 当代影响

## 观察区逻辑

历史事件用「关键节点与符号」展示，3–5 个：核心地理范围（示意地图）/ 标志性历史文物或图像 / 关键人物肖像式符号 / 重要场景示意 / 时间轴关键点。

用「地图」「文物」「人物」「场景」等短标题，不编号。横向事件时间轴是该类核心可视化，建议必含。

## 可视化条带

- 横向事件时间轴：线上标 4–6 个关键日期圆点
- 示意地图：标出核心区域或战区，配地名标签
- 因果链路图：用 3–5 个节点展示背景、爆发、转折与影响

## 一句话锚定

用户第一眼必须认为：这是一本历史百科里的事件页，客观梳理着这一事件的来龙去脉与影响。

而不是：历史题材电影海报、爱国宣传画、戏剧化插图。

## 防翻车规则

- 主视觉用史料图鉴风（地图/文物/示意图组合），不做战争大片、英雄主义海报
- 史学评价保持中立多元，呈现主流史观也提示不同视角，不单一立场说教
- 涉及战争伤亡等敏感数据，用学界约数范围，不编造精确数字、不渲染血腥
- 不美化或丑化任何一方，不写煽动性、立场性文案
- 时间轴用确定的历史共识年份，争议处注明「约」「据载」
- 不出现「永远铭记」「不可饶恕」等情绪化口号
