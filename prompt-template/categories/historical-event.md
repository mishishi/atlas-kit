# Historical Event Category Template

## Accent Color
* Antique Gold #C08040

## Identity
* Historical encyclopedia event page

## Hero Illustration
* Archival composition of map, artifact, people, and key scene
* 3–5 small historical-symbol or event-node callouts

## Information Modules
* 事件档案
* 背景起因
* 事件经过
* 核心数据
* 主要人物
* 结果影响
* 史学评价
* 快速评分卡

## Rating Dimensions
* 规模烈度
* 持续时长
* 波及范围
* 历史转折性
* 当代影响

## Observation Insets
* 地图
* 文物
* 人物
* 场景
* 节点

## Visualization Strip
* 1–2 compact visualizations
* Event timeline
* Regional map
* Cause-effect chain

## Failure Prevention
* No propaganda aesthetics
* No war-movie poster style
* No emotional slogans
* Maintain historical neutrality
