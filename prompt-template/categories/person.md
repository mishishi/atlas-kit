# 分类模板 · 人物

> 用法：接在主模板之后。用本模板的内容替换主模板中的主视觉逻辑、8 个模块栏目、评分维度，并叠加防翻车规则。

## 强调色

深紫 `#6B5EA8`

## 固定 8 模块

1. 基本档案
2. 时代背景
3. 生平脉络
4. 核心成就
5. 历史地位
6. 思想主张
7. 相关人物
8. 快速评分卡

## 评分维度

历史影响 / 时代代表性 / 创新程度 / 后世传播 / 争议程度

## 观察区逻辑

人物不用「身体局部放大」，放大脸或手既无信息量又容易画崩。

观察区放「与人物绑定的标志物 / 符号」3–5 个：代表作（书籍/画作/发明的剪影或图示）/ 标志性随身物或服饰 / 时代背景符号 / 关键场所 / 手稿签名印章式符号。

用「代表作」「时代」「手稿」等短标题，不编号。

## 可视化条带

- 紧凑横向生平时间轴：出生 → 3–4 个关键节点 → 逝世，标注年份
- 作品 / 成就关系图：用 3–5 个节点连接代表作、事件、人物或地点
- 行迹地图或关键地点示意图

## 一句话锚定

用户第一眼必须认为：这是一本人物百科里的档案页，肖像旁配着生平与成就的结构化梳理。

而不是：名人海报、颁奖典礼宣传图、个人崇拜式伟光正画像。

## 防翻车规则

- 主视觉肖像用图鉴化 / 版画 / 素描风，避免照片级写实人脸——既减少画崩风险，也更符合图鉴气质
- 历史人物不配现代证件照感的脸，不用穿越感的现代打光
- 不做成颁奖典礼、杂志封面、个人崇拜海报
- 评价保持百科中立：有争议者照实标注争议，不无脑歌颂也不丑化
- 关键语录 / 思想主张要可考，不编造名人名言
- 生卒年、关键事件用历史共识，模糊处用「约」「相传」
- 不出现「伟大」「传奇」「神级」等宣传赞美词
