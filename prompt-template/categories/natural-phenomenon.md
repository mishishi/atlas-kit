# 分类模板 · 自然现象

> 用法：接在主模板之后。用本模板的内容替换主模板中的主视觉逻辑、8 个模块栏目、评分维度，并叠加防翻车规则。

## 强调色

风暴蓝灰 `#4A7A9B`

## 固定 8 模块

1. 现象档案
2. 成因机制
3. 规模参数
4. 分布规律
5. 持续周期
6. 影响评估
7. 风险与应对
8. 快速评分卡

## 评分维度

破坏力 / 预测难度 / 发生频率 / 影响范围 / 研究成熟度

## 观察区逻辑

自然现象用「成因与形态分解」，3–5 个：形成机制关键环节 / 典型形态特征 / 不同阶段对比 / 规模或强度示意 / 前兆信号。

用「成因」「形态」「阶段」「前兆」等短标题，不编号。成因流程图（带方向箭头）是该类核心可视化，建议必含。

## 可视化条带

- 成因机制流程图（3–5 步，带方向箭头）
- 世界或区域地图，阴影标出高发区
- 结构剖面或能量流向示意图，展示关键物理过程

## 一句话锚定

用户第一眼必须认为：这是一本地理 / 气象科普图鉴里的现象页，用机制图解释它如何形成与演变。

而不是：灾难新闻大片、末日感渲染图、惊悚海报。

## 防翻车规则

- 不做成灾难新闻摄影、末日大片、惊悚渲染
- 主视觉是冷静的科学示意图，不渲染恐怖灾难场面
- 涉及灾害的影响 / 风险，客观分级陈述，不煽动恐慌、不渲染惨状
- 成因机制求科学准确，不简化成错误因果
- 规模参数用量级范围（如风速等级、震级范围），不编精确数值
- 风险等级作科普示意，不冒充权威预警发布
