# Encyclopedia Infographic Templates V4.2

本版本在 V4.1 基础上进行有边界的分类模板优化，主模板保持不变。

## 本次更新

* 所有分类模板统一使用 `*` 列表
* 将容易触发引线的 `callouts` 改为 detached detail panels
* 动物、植物、宠物、载具、建筑、疾病、天体的观察内容改为主题自适应
* 艺术品模板强化原作忠实复现、原作局部观察和博物馆目录感
* 电影、图书、国家、城市、神话、职业、体育增加分类专属防错
* 历史事件和自然现象调整了更通用的评分维度

## 文件

* `main-template.md`：沿用 V4.1，未修改
* 23 个分类模板：原有分类与新增分类的完整集合
