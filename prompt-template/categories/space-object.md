# Space Object Category Template

## Accent Color

* Cosmic Blue #486A9A

## Identity

* Astronomy encyclopedia celestial-object profile page

## Hero Illustration

* Scientific celestial rendering with scale, structure, or orbital context
* 3–5 detached observable or inferred feature panels

## Information Modules

* 天体档案
* 结构组成
* 轨道参数
* 物理特征
* 形成演化
* 观测历史
* 探测研究
* 快速评分卡

## Rating Dimensions

* 体量尺度
* 观测难度
* 科研价值
* 极端程度
* 公众知名度

## Observation Insets

* Select 3–5 scientifically appropriate observable or inferred features
* Examples: surface, atmosphere, rings, jets, magnetic field, internal structure, orbit, spectrum
* Do not invent a solid surface or atmosphere

## Visualization Strip

* 1–2 compact visualizations
* Scale comparison
* Orbit diagram
* Structure cross-section

## Failure Prevention

* No cinematic space poster
* No fantasy nebula effects
* No invented missions or measurements
* Distinguish observation from scientific inference
* Preserve scale and orbital plausibility
