# 分类模板 · 节日

> 用法：接在主模板之后。用本模板的内容替换主模板中的主视觉逻辑、8 个模块栏目、评分维度，并叠加防翻车规则。

## 强调色

随节日文化定：取该节日最具代表性的传统主色调。

通用规则：喜庆 / 团圆类节日用暖色（朱红、金黄、橙）；肃穆 / 纪念类节日用沉稳色（藏蓝、墨绿、暗金）；自然 / 季节类节日用对应季节色（春绿、秋金等）。优先采用该节日在其文化中约定俗成的标志色，而非随意选取。

参考示例：春节 → 朱红 `#C0392B` · 圣诞 → 深绿 `#2E7D32` · 开斋节 → 金绿 `#8D6E0A` · 中秋 → 暖金 `#C99A2E` · 万圣节 → 南瓜橙 `#D4691E` · 清明 → 青墨 `#4A6B5A`

## 固定 8 模块

1. 节日档案
2. 起源传说
3. 核心习俗
4. 代表食物
5. 代表服饰
6. 文化意涵
7. 现代演变
8. 快速评分卡

## 评分维度

文化重要性 / 传统保留度 / 全球知名度 / 参与门槛 / 商业化程度

## 观察区逻辑

节日用「习俗符号要素」展示，3–5 个：标志性器物（如灯笼/彩蛋/月饼）/ 代表色彩与图案 / 典型场景（团圆/游行/祭祀）/ 特色服饰元素 / 仪式动作。

用「器物」「色彩」「场景」「服饰」等短标题，不编号。

## 可视化条带

- 简易日历指示图，标出节日在一年中的位置
- 习俗流程图：用 3–5 个图标串联核心仪式或活动
- 节日符号带：展示代表器物、食物、服饰或图案

## 一句话锚定

用户第一眼必须认为：这是一本民俗文化图鉴里的节日页，梳理着这个节日的起源、习俗与意涵。

而不是：节日促销海报、电商大促页、贺卡。

## 防翻车规则

- 不做成节日促销、电商大促、商场氛围广告
- 不出现「狂欢」「钜惠」「限时」等营销词
- 起源传说作为文化知识陈述，不写成煽情故事会
- 习俗呈现地域多样性，不把单一地区习俗当作唯一标准
- 宗教 / 民俗背景客观陈述，保持文化中立尊重
- 不堆砌具体年份，起源用「相传」「约」等表述
