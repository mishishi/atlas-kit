# 分类模板 · 宠物

> 用法：接在主模板之后。用本模板的内容替换主模板中的主视觉逻辑、8 个模块栏目、评分维度，并叠加防翻车规则。

## 强调色

粉橙 `#E8845A`

## 固定 8 模块

1. 品种档案
2. 体型外貌
3. 性格特点
4. 饲养要点
5. 健康数据
6. 训练适性
7. 适合人群
8. 快速评分卡

## 评分维度

活跃度 / 亲人度 / 独立性 / 护卫性 / 训练难度

## 观察区逻辑

宠物用「品种特征放大」，重点是这个品种区别于其他品种的辨识点：耳型 / 被毛纹理与花色 / 口鼻吻部 / 爪型或体态 / 尾型。

用「耳型」「被毛」「吻部」等短标题，不编号，不连线。

## 可视化条带

- 体型剪影对比：品种与人手或人形并排，标注体重 / 体高
- 被毛 / 体态特征示意：用 3–5 个小图标展示品种识别点
- 护理周期小图表：标出梳毛、运动、健康检查等频率

## 一句话锚定

用户第一眼必须认为：这是一本犬猫品种图鉴里的品种档案页，像育种手册中的标准品种记录。

而不是：萌宠卖萌写真、宠物店广告、领养募捐海报。

## 防翻车规则

- 宠物允许适度亲和感（毕竟是伴侣动物），但仍是「品种标准展示」，不是卖萌摆拍
- 不做成宠物店广告、领养公益海报、社交媒体晒宠风
- 主视觉呈现品种标准外貌（标准站姿/坐姿），不要夸张表情、穿衣打扮、拟人道具
- 不出现「主子」「毛孩子」「铲屎官」等网络宠物用语
- 性格 / 评分只作品种倾向示意，注明因个体而异，不绝对化
- 健康数据用常识性范围，不编造精确患病率
