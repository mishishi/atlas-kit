# 分类模板 · 城市

> 用法：接在主模板之后。用本模板的内容替换主模板中的主视觉逻辑、8 个模块栏目、评分维度，并叠加防翻车规则。

## 强调色

钢蓝 `#3A7EBF`

## 固定 8 模块

1. 基础档案
2. 地理位置
3. 历史沿革
4. 城市结构
5. 代表地标
6. 美食文化
7. 生活方式
8. 快速评分卡

补充模块（空间允许时）：地图速览 · 城市印章 · 地标剪影 · 城市时间轴

## 评分维度

历史底蕴 / 城市识别度 / 宜居指数 / 交通便利 / 文化特色

## 观察区逻辑

城市不用「身体局部放大」，城市没有可解剖的器官。

主视觉应像百科城市图鉴的城市总览：能同时看出城市地貌、核心地标、街区肌理和历史气质。

观察区展示 3–5 个城市要素：核心地标 / 城市地貌 / 历史符号 / 街区肌理 / 生活场景。可用小插图或剪影，不与主视觉精确连线，用「地标」「地貌」「街区」「生活」「历史符号」等短标题，不用数字编号。观察图像百科页的知识补充，不是游玩推荐标签。

标题区右侧可加入极淡的城市剪影：选择该城市最有识别度的 3–5 个地标、天际线或城墙轮廓，用低对比度线稿呈现，像书页上的浅色档案插图，不要做成照片拼贴或醒目装饰。

底部总结栏要像百科书页的精致收尾：一句沉稳的城市判断居中，左右可放城市门楼、地形线、地标小剪影或城市印章式图形；装饰数量少、线条细、颜色浅，保留 v2 的清爽网格感。

## 可视化条带

- 小型示意地图：标出城市的地理位置，圆点标记，只做简洁示意
- 城市时间轴：用 4–5 个阶段标签概括城市历史脉络
- 地标剪影带：选 3–5 个代表地标做低对比度剪影

## 一句话锚定

用户第一眼必须认为：这是一本百科全书中的城市图鉴页。

而不是：城市广告。

## 防翻车规则

- 不做成城市宣传拼贴、地标照片墙、明信片页面或商业广告
- 不出现「必游」「打卡」「网红」「攻略」「路线推荐」「魔都」等游玩或营销语气
- 代表地标用剪影、局部图标或小插图，不要把所有地标挤进主视觉
- 地图速览只做简洁示意，不画复杂路线、精确坐标或伪地铁图
- 城市时间轴 / 历史沿革只用阶段性标签，不堆砌过多具体年份
- 不混入其他城市的标志性地标
- 评分卡只作主观体验示意，不伪装成权威城市排名
- 主视觉冷静克制，不要戏剧化光影、不要旅游大片感
