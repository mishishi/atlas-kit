# 分类模板 · 器物

> 用法：接在主模板之后。用本模板的内容替换主模板中的主视觉逻辑、8 个模块栏目、评分维度，并叠加防翻车规则。

## 强调色

石板银 `#6A7E8A`

## 固定 8 模块

1. 器物档案
2. 结构组成
3. 工作原理
4. 材质工艺
5. 关键参数
6. 发展历程
7. 使用场景
8. 快速评分卡

## 评分维度

制作精密度 / 历史重要性 / 文化象征性 / 使用普及度 / 收藏价值

## 观察区逻辑

器物最适合「爆炸视图 / 剖面拆解」——这是器物图鉴最强的表达。

观察区放 3–5 个：关键部件特写 / 内部结构剖面 / 材质纹理 / 功能接口或连接处 / 操作部位。用「部件」「剖面」「材质」等短标题，不编号。爆炸视图或剖面图建议作为主视觉或核心可视化。

## 可视化条带

- 爆炸视图或剖面结构图，标注主要部件
- 代际演变时间轴：展示早期形态到典型形态的变化
- 使用场景小图标带：标出典型用途或操作环境

## 一句话锚定

用户第一眼必须认为：这是一本器物 / 工业设计图鉴里的物件页，像工程图纸般拆解它的结构与原理。

而不是：商品销售页、电商主图、产品广告。

## 防翻车规则

- 不做成电商商品图、产品广告、开箱评测风
- 主视觉用图鉴 / 工程制图风，精确呈现结构，不做唯美产品摄影
- 不出现「爆款」「热销」「必入」等电商语
- 结构拆解求准确，部件标注清楚，不画看着复杂但错误的伪结构
- 参数用常识性量级，不编造精确规格
- 发展历程用代际演变，不堆砌具体型号年份
