# 分类模板 · 动物

> 用法：接在主模板之后。用本模板的内容替换主模板中的主视觉逻辑、8 个模块栏目、评分维度，并叠加防翻车规则。

## 强调色

暖琥珀 `#E07840`

## 固定 8 模块

1. 分类阶元
2. 形态特征
3. 栖息环境
4. 食性行为
5. 繁殖习性
6. 保护现状
7. 分布地图
8. 快速评分卡

## 评分维度

体型 / 攻击性 / 稀有度 / 适应力 / 人类熟悉度

## 观察区逻辑

动物可以用「身体局部放大」——这是它区别于城市的地方，动物有值得解剖式观察的部位。

观察区展示 3–5 个最具辨识度或功能性的局部：头部特征（眼/耳/角/喙）/ 爪蹄鳍等运动器官 / 皮毛鳞片羽毛纹理 / 牙齿口器 / 尾部。

优先选该物种区别于近亲的关键特征，用「头部」「前爪」「皮毛」等短标题，不用编号，不连线。

## 可视化条带

- 小型世界地图，阴影标出该物种自然分布区
- 体型对比剪影：动物与人类轮廓并排，带比例尺
- 生态位示意：用小图标标出栖息地、食性与活动时间

## 一句话锚定

用户第一眼必须认为：这是一本自然博物图鉴里的物种观察页，像博物学家手绘标本旁的记录。

而不是：萌宠写真、动物摄影大片、动物园宣传海报。

## 防翻车规则

- 不卡通化、吉祥物化、拟人化，即使是大熊猫、猫狗等自带可爱属性的物种
- 不做野生动物摄影海报——避免戏剧光影、逆光剪影、电影感大片构图
- 主视觉是冷静的标本式观察图，不是情绪化抓拍
- 保护现状只做客观分级（如 IUCN 等级），不配煽情文案、不做公益募捐风
- 不出现「萌」「治愈」「可爱」「呆」等情绪化标签
- 分布地图只做大范围示意，不画精确栖息坐标
- 数据用范围或约数（体长「1–1.3 米」），不编精确小数统计值
